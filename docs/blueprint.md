# **App Name**: Elite Access

## Core Features:

- Persuasive Content Display: Render the key marketing messages including the urgent headline, subheadline, support information, and urgency alerts.
- Limited Access Badge: Prominently display the 'ACESSO LIMITADO' badge at the top of the page, establishing exclusivity.
- Urgency Alert System: Highlight a dynamic alert box, e.g., '⚠️ Somente 100 testes liberados hoje.', to create scarcity and drive immediate action.
- Unmissable WhatsApp Call-to-Action: Implement an oversized, distinctive green WhatsApp button that links directly to the specified contact for seamless communication.
- Mobile-First Responsive Layout: Ensure the entire landing page and all its elements are fully responsive, optimizing the visual impact and user experience across all devices, especially mobile.

## Style Guidelines:

- Dark premium scheme: The background utilizes a very dark, subtle purplish-red hue (#221A1D), serving as a base for modern gradients to convey sophistication and depth.
- Primary Color: A rich crimson (#D32B50) will be used for dominant text elements and strategic accents, symbolizing urgency and exclusivity while providing strong contrast on the dark background.
- Accent Color: A vibrant orange-red (#E66E3A) will highlight interactive elements and create soft, technological glow effects, intensifying the aggressive and modern aesthetic.
- WhatsApp Button Color: A bright, standard WhatsApp green (#25D366) will ensure the main call-to-action button stands out powerfully against the dark and red thematic elements.
- Headline and body font: 'Space Grotesk' (sans-serif) for its modern, techy, and strong character, perfect for large, impactful headlines and concise body text with high visual contrast.
- Integrate the unmistakable WhatsApp logo within the oversized CTA button, and use clear, impactful icons like the warning '⚠️' emoji to emphasize key messages.
- Centralized and responsive layout: All key information and the massive WhatsApp button will be centrally aligned for direct focus, with a design optimized for an imposing mobile presentation.
- Subtle red glow: Incorporate soft, red glowing effects around critical elements like the urgency alert box and the CTA button, reinforcing the premium, urgent, and technological feel.
- CTA interaction: Implement dynamic hover animations on the WhatsApp button, such as a slight scale-up or intensified glow, to strongly encourage visitor engagement.
